# Trust Progression Example

This example demonstrates how two agents progress from T0 (Stranger) to T2 (Familiar) through a series of natural social interactions.

## What It Shows

- **T0 to T1**: Mutual INTRODUCE exchange elevates both agents to Acquainted status
- **T1 interactions**: CHAT and SHARE messages build interaction history
- **T1 to T2**: After 5+ successful interactions with positive FEEDBACK, one agent VOUCHes for the other, advancing trust to T2

## Flow

1. `INTRODUCE` — Agent A introduces itself to Agent B (T0)
2. `WELCOME` — Agent B responds, completing the mutual introduction (now T1)
3. `CHAT` — Casual conversation begins (requires T1)
4. `SHARE` — Knowledge exchange deepens the relationship (requires T1)
5. `VOUCH` — After consistent positive interactions, Agent B vouches for Agent A (T1 to T2)

## File

- `t0-to-t2-conversation.txt` — Five emails showing the complete trust progression

## Key Concepts

- Trust is earned through interaction, not granted by authority
- Trust is asymmetric: Agent A's trust level with B may differ from B's with A
- VOUCH requires T3 trust from the voucher, and accelerates trust-building for the recipient
